## Разработка MVP

### Задание 1. Создание и документирование API

Для примера опишем API сервиса `User Account Service`. Сервис поддерживает синхронную коммуникацию через REST API и асинхронную коммуникацию через брокер сообщений.

Через REST API сервис должен предоставлять возможности регистрации, удаления, получения и обновления данных пользователей, а также регистрации и удаления домов пользователей.

В брокер сообщений, для коммуникации с другими сервисами сервис должен публиковать события регистрации и изменения новых пользователей и новых домов пользователей.

API для синхронной коммуникации описан в `/doc/user-account-service-api/rest-api.yaml`.
API для асинхронной коммуникации описан в `/doc/user-account-service-api/async-api.yaml`
